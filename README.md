# Kitchen English

App web minimalista para aprender vocabulario de cocina en inglés.

## Incluye

- 85 verbos de cocina.
- 80 utensilios de cocina.
- Light mode, dark mode y modo automático.
- Flashcards con animación flip.
- Botones `I know it`, `I don't know` y `✦ Details`.
- Búsqueda, filtros por categoría y dificultad.
- Progreso con palabras vistas, aprendidas, porcentaje y racha.
- EXP, niveles y objetivos.
- Quiz final.
- Pronunciación usando Text-to-Speech del navegador.
- PWA básica: manifest + service worker.

## Cómo abrir localmente

Solo abre `index.html` en el navegador.

## Cómo subir a Vercel

1. Sube estos archivos a un repositorio de GitHub.
2. En Vercel, elige **Add New → Project**.
3. Importa el repositorio.
4. Framework Preset: **Other**.
5. Root Directory: `./`.
6. Deploy.

No ocupa instalar dependencias.
